# Tabs
## Tabs Summary
Tabs are an organizational method where the user can create multiple tabs at the top of the Symbiote window. These tabs can further organize dissimilar things as the user wishes. Example, Tab 1 can be for stats, saving throws, and skills like Crafting, Investigation, Survival, and Nature. Then on Tab 2, the user can put their inventory of items. And finally Tab 3 can be their notes section to store simple notes about their character and recent events. These tabs can all be renamed should the user want to. Tabs can also be recolored from a few simple colors, or via RGBA picker that's provided.

## Tab Names
By default, Character Vault has one default tab with a default name of `Tab 1`. The user can rename this tab via `Edit` mode.

## Reorder Tabs
By clicking & dragging, the user can change the order of the tabs. By default, they are in the order in which they're created.

## Scrolling Tabs
The tabs bar can scroll side-to-side to accomodate if the user creates more tabs than will fit on the screen at one time.

## Deleting Tabs
In `Edit` mode, the user has the option of deleting a tab. Using a modal window, confirm with the user before deleting a tab.
