# Character Vault

## Summary
Character Vault is best described as a "Character Sheet Engine". It is a tool to craft and customize character sheets that can serve any roleplaying system as it does not bind itself to any particular ruleset. Want to build your own 5e character statblock, or playing an RPG system you cooked up at home? Character Vault can help you bring that into Talespire.

## What it is NOT
Character Vault is a Swiss-Army Knife of character sheets. That means it will not automate much, if anything, in terms of game rulesets like increasing a stat will automatically increase your modifier. Character Vault is not designed for that.
